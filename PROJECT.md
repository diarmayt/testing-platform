 PROJECT: Платформа тестирования (MVP) → позже курсы и монетизация

## 1) Цель MVP
Сделать рабочий сайт тестирования:
- регистрация только по приглашению
- студент проходит тест: один вопрос → следующий
- результат + разбор (включается/выключается в админке)
- интерфейс RU/EN/KZ
- хороший современный дизайн в стиле “cosmic gradient”

## 2) Будущее (не делаем сейчас)
- Курсы как LMS (Course → Module → Lesson)
- Монетизация (организации, планы, лимиты)
- Anti-cheat (lite → потом прокторинг/камера)

## 3) Стек (фиксируем)
- Next.js (App Router) + TypeScript
- Tailwind + shadcn/ui (крутой дизайн)
- next-intl (RU/EN/KK)
- Postgres + Prisma
- Auth.js (credentials сначала, Google потом)
- Docker + Docker Compose
- GitHub Actions (CI)
- pnpm

## 4) Правила работы (самое важное)
- Пишем и объясняем всё на русском (кроме кода/команд).
- Делаем маленькие шаги: 1 шаг = 1–3 коммита.
- После каждого шага Antigravity обязан выдать:
  1) что сделано
  2) какие файлы изменены
  3) команды запуска/проверки
  4) QA чеклист (что кликнуть, чтобы найти баги)
  5) commit message
- main всегда “зелёный”: проект не ломаем.
